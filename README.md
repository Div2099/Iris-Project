# 🌸 Iris Dataset Data Science Project (Individual Project)

## 📌 Project Overview

This is an individual data science project based on the **Iris dataset**, aimed at performing data analysis and building machine learning models to classify iris flowers into different species.

The project demonstrates the complete data science workflow, including data preprocessing, visualization, model building, and evaluation.

---

## 📊 Dataset Information

The project uses the **Iris Dataset**, a widely used dataset for classification tasks.

### Features:

* Sepal Length (cm)
* Sepal Width (cm)
* Petal Length (cm)
* Petal Width (cm)

### Target Classes:

* Setosa
* Versicolor
* Virginica

---

## 🎯 Objectives

* Analyze and understand the dataset
* Visualize relationships between features
* Build and compare classification models
* Evaluate model performance

---

  * scikit-learn

## 🔍 Methodology

### 1. Data Loading

* Loaded dataset using pandas / sklearn

### 2. Data Exploration

* Checked dataset structure and summary statistics
* Verified absence of missing values

### 3. Data Visualization

* Used pair plots and histograms
* Generated correlation heatmap

### 4. Model Building

* Split dataset into training and testing sets
* Applied machine learning algorithms:

  * Logistic Regression
  * K-Nearest Neighbors (KNN)
  * Decision Tree

### 5. Model Evaluation

* Evaluated models using:

  * Accuracy Score
  * Confusion Matrix
  * Classification Report

---

## 📈 Results

* Achieved high accuracy in classifying iris species
* Found petal length and petal width to be strong predictors
